# 对象数组
> 就是数组里的每个元素都是类的对象，赋值时先定义对象，然后将对象直接赋给数组就行了。

<!-- more -->

**Student类：**
```
public class Student {
	// 成员变量
	private String name;
	private int age;

	// 构造方法
	public Student() {
		super();
	}

	public Student(String name, int age) {
		super();
		this.name = name;
		this.age = age;
	}

	// 成员方法
	// getXxx()/setXxx()
	public String getName() {
		return name;
	}

	public void setName(String name) {
		this.name = name;
	}

	public int getAge() {
		return age;
	}

	public void setAge(int age) {
		this.age = age;
	}

	@Override
	public String toString() {
		return "Student [name=" + name + ", age=" + age + "]";
	}
```

## 遍历对象数组

### 普通for
```
for (int x = 0; x < students.length; x++) {
	Student s = students[x];
	System.out.println(s.getName()+"---"+s.getAge());

	//System.out.println(students[x]);	//调用toString();
}
```

### 增强for
```
for(Student s : students){
	System.out.println(s.getName()+"---"+s.getAge());
	System.out.println(s);
}
```

#### 二者区别：
- 增强for:
	- 简洁
	- 不能指定范围访问，只能访问全部
	- 不能逆序（反转）输出
	- 不能修改元素的值
